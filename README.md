# APSA
Automation Platforms Strategy &amp; Architecture Tools.
An apple style web page for navigation around the tools.

## How To Use
A merge or commit to main branch will trigger hook to the apiServer running on the host which will pull the changes down.
Requires a web server running on the host for the web page to be hosted on http://lonasdbrun02.uk.db.com: 800

### ap_tech-radar-master
A technology radar for our products, languages, techniques and tools in AP

### ap_apiServer
A flask API server for whatever you need, but needs to be running for deployments of APSA tools to run

### ap_airflow
Contains dags for running APSA automation
